# Interaction GAM Benchmark

Uses curated pediatric DDI set for real-world validation of GAM method

Downstream project, consumes output from:
 `ddi_reference_set/`.

## Execution order

From `gam_benchmark/`:

```powershell
R scripts\R\01_validate_pediatric_ddi_benchmark.R
# Optional, generates one graph per positive triplet
R scripts\R\02_positive_triplet_graphs.R
```

## Structure

- `00_functions.R`: contains model functions (GAM, IOR/AC), global      configuration of the pipeline, data loading and mapping
- `scripts/R/01_validate_pediatric_ddi_benchmark.R`: maps curated dataset with `ade_raw` identifiers. Model fitting and performance metrics
- `scripts/R/02_positive_triplet_graphs.R`: mimics `gam_validation/40_graphs.R` to generate one graph for every positive tripllet.
- `data/processed/`: solo `.gitkeep`; `ade_raw.csv` se lee en vivo desde `faers_parsing/`.
- `results/`: output.

## Input



- `../faers_parsing/data/processed/ade_raw.csv` - generated by `faers_parsing/`, read live (no local copy).
- `../data/vocabulary/vocabulary_SNOMED_MEDDRA_RxNorm_ATC/CONCEPT.csv` - shared OMOP vocabulary.
- `../data/vocabulary/vocabulary_SNOMED_MEDDRA_RxNorm_ATC/CONCEPT_ANCESTOR.csv`
- `../ddi_reference_set/results/curated_pediatric_ddi_reference_set/curated_pediatric_ddi_triplets.csv` - curated real-world IDD pediatric dataset.
- `results/sics/null_distribution_results/null_thresholds*.csv` - null tresholds from `gam_validation/`.
